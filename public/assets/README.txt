========================================
IMPORTANT: ADD YOUR FILES TO THIS FOLDER
========================================

This folder is for your personal assets that will be displayed on your portfolio website.

REQUIRED FILES:
---------------

1. **resume.pdf**
   - Your resume/CV in PDF format
   - Will be downloadable via "Download Resume" buttons on the website
   - Naming: Must be exactly "resume.pdf" (lowercase)
   - Example: Copy your "MY Resume.pdf" and rename it to "resume.pdf"

2. **profile.jpg** or **profile.png**
   - Your professional profile photo
   - Will be displayed in the Hero section (top of the page)
   - Naming: Must be exactly "profile.jpg" or "profile.png" (lowercase)
   - Recommended: Square photo, at least 500x500px for best quality

HOW TO ADD FILES:
-----------------

1. Copy your resume PDF to this folder
2. Rename it to: resume.pdf
3. Copy your profile photo to this folder  
4. Rename it to: profile.jpg (or profile.png)
5. Refresh your browser to see the changes

FILE LOCATIONS IN PORTFOLIO:
-----------------------------

✓ Profile Picture: Hero section (top of page, animated circle)
✓ Resume Download: 
  - Hero section ("Download Resume" button)
  - About section ("Download Resume" button)

CURRENT FILE STRUCTURE:
-----------------------

  ├── README.txt (this file)
  ├── resume.pdf (ADD YOUR FILE HERE)
  └── profile.jpg (ADD YOUR FILE HERE)

NOTE: If you don't add these files, the website will still work:
- Profile photo will show "SS" initials instead
- Resume buttons will show but won't download anything until you add the file
